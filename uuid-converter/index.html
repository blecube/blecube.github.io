128-bit UUID Converter

First use nRFgo Studio to create a custom 128-bit UUID in ASCII format. Use this tool to convert from ASCII to Hex representation for use in embedded C with nRF51 and nRF52 development kits.

The ascii UUID must be in the following format:
A0B0xxxx-C0D0-E0F0-A1B1-C1D1E1F1A2B2
